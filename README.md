Fashion AI

Что это делает

Загружаешь фотографию с одеждой, система находит предметы и советует, что с чем носить. Работает через веб-интерфейс или API.

Как работает

1. YOLOv8 ищет людей на фото
2. CLIP классифицирует одежду по зонам тела (верх, низ, обувь, аксессуары)
3. Правила на основе стиля (casual, formal, sporty) формируют рекомендации
4. Показывает что есть на фото и чего не хватает для полного образа

Что реально используется

- YOLOv8n для поиска людей
- CLIP (openai/clip-vit-base-patch32) для классификации
- FastAPI для бэкенда
- Gradio для простого интерфейса
- OpenAI API (опционально) для текстовых советов

Как запустить

1. Установить зависимости:
   pip install -r requirements.txt

2. Запустить:
   python run.py

3. Открыть http://localhost:7860 в браузере

Если нужен только API без интерфейса:
   python run.py --api-only

Что есть сейчас

- Детекция одежды на фото с человеком
- Классификация по категориям
- Рекомендации по стилям (casal, formal, sporty)
- Учет пола для рекомендаций (male/female/auto)
- Аннотированные изображения
- Web UI на Gradio
- REST API

- Интеграция с магазинами (ссылки на похожие товары)

Известные проблемы

- Иногда путает похожие предметы (например, блузку и рубашку)
- На групповых фото работает нестабильно
- Требует нормального освещения

Структура файлов

app/
  detection.py      - логика YOLO + CLIP
  recommendations.py - правила рекомендаций
  main.py           - FastAPI endpoints
  gradio_app.py     - веб-интерфейс
  llm_helper.py     - генерация текстовых советов

Требования

- Python 3.10+
- 4GB+ RAM
- GPU не обязателен, но ускоряет CLIP
